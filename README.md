#Challenge 1:Grade Generator

This is a simple JavaScript program that evaluates a student's marks and assigns a corresponding grade based on predefined grade boundaries.

Usage
Open the file grade_.js in a JavaScript environment.
Modify the value of the variable marks to reflect the student's actual marks.
Run the program.
The program will output the corresponding grade based on the provided marks.
Grade Boundaries
A: 79 - 100
B: 60 - 78
C: 49 - 59
D: 40 - 48
E: 0 - 39
Example
If marks is set to 45, the program will output:

mathematica

D
indicating that the student's grade is a "D".

#salary calculator
Salary Calculation Program
This JavaScript program calculates the net salary of an employee based on their basic salary and benefits, while considering statutory deductions such as PAYE, NHIF, and NSSF.

Usage
Open the file salary_.js in a JavaScript environment.
Run the program.
Input the required information when prompted:
Basic salary: Enter the employee's basic salary.
Benefits: Enter any additional benefits received by the employee.
The program will calculate and display the net salary after deductions.
Deduction Rates
PAYE (Pay As You Earn) Rate: 30% of the basic salary
NHIF (National Hospital Insurance Fund) Rate: 1.5% of the basic salary
NSSF (National Social Security Fund) Rate: 6% of the basic salary
Example
For example, if the basic salary is entered as $2000 and benefits as $500, the program will calculate the net salary as follows:

swift

Your net salary is $1570
Note
This program assumes that the user provides valid numeric inputs for basic salary and benefits.
The net salary calculation does not account for any other deductions or allowances apart from PAYE, NHIF, and NSSF.
